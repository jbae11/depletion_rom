\title{Deep Learning Approach to Nuclear Fuel Transmutation in a Fuel Cycle Simulator}

\section*{Highlights}
\begin{itemize}
       \item Usage of deep neural networks to predict depleted fuel composition reduces computational time while increasing accuracy.
       \item The trained neural network predicts used PWR assembly
             decay heat and activity with less than 5\% error.
       \item The trained neural network model can be easily distributed
             since it provides no information on the original data. 
\end{itemize}
